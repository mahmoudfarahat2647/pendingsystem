Eye Icon Placement — Which approach fits your auth style?

Three options for adding password visibility toggle to the fieldset-legend input design

Password

Icon sits inside the fieldset border, right side

A — Inside Fieldset

Eye icon lives within the fieldset border alongside the input. Feels contained, tight, and clean. Matches the fieldset layout naturally.

Password

Icon floats on the fieldset border, top-right

B — Floating on Border

Icon is positioned on the top border of the fieldset like a second legend. Unconventional and stylish, but may clash with the existing legend label.

Password

Icon absolutely positioned over fieldset, muted white

C — Absolute Overlay (muted)

Icon overlays the fieldset absolutely, using muted white color. Subtle and standard. Activates to full yellow on hover. Most common pattern.

Context: The auth card uses dark #0A0A0A/60 background with #FFCC00 yellow accents. Password field uses a fieldset+legend pattern with a yellow border.
